# Aprendendo Go 🐹

Repositório de estudos da linguagem **Go (Golang)**, do zero ao avançado.

## 📚 Tópicos Estudados

### ✅ Nível 1: Fundamentos
- Hello World & Estrutura de arquivos
- Variáveis e Tipos Básicos
- Estruturas de Controle (`if`, `for`, `switch`)
- Funções com retornos múltiplos

### ✅ Nível 2: Estruturas de Dados
- Arrays e Slices
- Maps (com padrão "Comma ok")
- Structs

### ✅ Nível 3: OOP no estilo Go
- Métodos e Interfaces
- Composição sobre Herança
- Ponteiros

### 🔄 Nível 4: Concorrência (Em andamento)
- Goroutines
- Channels
- WaitGroups

## 🚀 Como rodar

```bash
go run main.go
```

## 🛠 Ambiente

- Go 1.26.2
- macOS
